# Guia de Desenvolvimento

## Requisitos

- Java 11 ou superior instalado

## Execução Local

Rode diretamente o executável já compilado:

```bash
java -jar the-lost-hero.jar
```

Ou compile o código fonte em `src/` com seu IDE Java de preferência (Main.java é o ponto de entrada).

## Para Desenvolvedores

- Código limpo e bem documentado
- Arquitetura modular e extensível
- Sistema de recursos dual (desenvolvimento + JAR)
- Logs detalhados para debug
- Tratamento de exceções abrangente
